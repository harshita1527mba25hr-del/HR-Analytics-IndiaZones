# HR-Analytics-IndiaZones
##  Overview
This repository contains HR analytics data for employees across different zones in India. It provides insights into attrition, salaries, bonuses, performance ratings, and workforce distribution. The dataset is designed to support HR dashboards, predictive modeling, and AI-powered workforce transformation projects.

##  Dataset Details
- **HR_Data_IndiaZones.csv** → Employee-level HR data including demographics, salaries, bonuses, attrition, and departmental information.  
- **india-states.json** → Geospatial mapping data for Indian states, useful for visualizing HR metrics by region.  

##  Key Features
- Employee attrition analysis by zone and department  
- Salary and bonus distribution across roles and regions  
- Gender-based workforce distribution  
- Performance rating trends  
- Geospatial visualization of HR metrics  

##  Tools Used
- **Power BI** → For interactive dashboards and visualizations  
- **Python (Pandas, Matplotlib, Scikit-learn)** → For data analysis, visualization, and machine learning modeling  
- **Faker** → For synthetic dataset generation
